## Story Template

Each user story follows this structure:

```markdown
# User Story: [Title]

**ID:** STORY-XXX

## User Story Statement
As a [user type], I want to [action], so that [benefit].

## Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2

## Related Entities
- Links to domain model

## Related Flows
- Links to workflows

## Acceptance Tests
Gherkin scenarios

## Definition of Done
- [ ] Checklist item
```
